# HW02 — Базовый EDA(разведка), но с душой

В рамках данного домашнего задания был проведён первичный анализ табличного датасета с использованием библиотеки Pandas, а также выполнена базовая визуализация данных в Matplotlib.

## Что было сделано:

1. Загружен учебный датасет в pandas.DataFrame.
2. Выполнен первичный осмотр данных:
   - выведены первые строки;
   - изучена структура таблицы;
   - рассчитаны базовые статистики.
3. Проведен контроль качества данных:
   - рассчитаны пропуски;
   - найдены дубликаты;
   - выполнен поиск подозрительных (отрицательных) значений.
4. Выполнен минимальный EDA:
   - рассчитаны частоты одной категориальной переменной;
   - выполнена группировка с расчетом средних значений.
5. Построены 3 графика:
   - гистограмма;
   - boxplot;
   - scatter plot.
6. Все графики сохранены в папку `figures/`.

## Неформально о проделанной работе:

Данные были аккуратно продегустированы, градусы распределены, выбросы выявлены, а подозрительные значения отправлены на переосмысление. Анализ проведен трезво, но с уважением к предметной области.

Работа выполнена строго по ТЗ, но с лёгким творческим послевкусием.
